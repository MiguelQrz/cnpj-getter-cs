# cnpj-getter-cs
🧾 CLI de Consulta de CNPJ (.NET)

Aplicação de linha de comando desenvolvida em C# com .NET para consultar dados públicos de empresas brasileiras a partir do CNPJ.

🚀 Funcionalidades
Consulta de dados de empresas via CNPJ
Exibição formatada no terminal
Validação de CNPJ
Suporte a CNPJ com ou sem máscara
Integração com API pública de dados (ex: BrasilAPI, ReceitaWS)
🛠️ Tecnologias Utilizadas
.NET 10
C#
API -> https://opencnpj.org/
HttpClient
System.Text.Json
